# Learning Source of Sonoma County - Dev

Development environment for Learning Source of Sonoma County. Forked from VVV and modified to suit.

* **Version**: 1.2-lssc
* **Contributors**: [@randybuildsthings](https://github.com/randybuildsthings)

## Credentials and Such

All database usernames and passwords for WordPress installations included by default are `wp` and `wp`.

All WordPress admin usernames and passwords for WordPress installations included by default are `admin` and `password`.

### Base Development
* LOCAL PATH: vagrant-local/www/lssc-dev
* VM PATH: /srv/www/lssc-dev
* URL: `http://wordpress.lssc.dev`
* DB Name: `lssc`

### MySQL Root
* User: `root`
* Pass: `root`
* See: [Connecting to MySQL](https://github.com/varying-vagrant-vagrants/vvv/wiki/Connecting-to-MySQL) from your local machine
